Please follow the steps:

#1	Install Node 						https://nodejs.org/download/

#2	In the projects directory execute:	npm install

#3	In the projects directory execute:	npm link

#4	The wordcount command gets two arguments
		1st:	The type of information that the user needs wordcount basic, frequencies and extended.
		2nd:	The file location or the directory.

		eg. execute wordcount --basic sample.txt

#5 The results are going to be printed in the console and a log file will be created in the logs directory
		wordCountLogs
